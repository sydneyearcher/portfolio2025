My Work

Exercise App in React

In this project, I developed an interactive exercise application using React. The app features a dynamic menu system where the display of components is controlled through ternary operators within JSX elements. This approach allows for conditional rendering based on user interactions, enhancing the app's responsiveness and user experience. The project demonstrates proficiency in React's state management and conditional rendering techniques.

Object Oriented Design

This project showcases the application of object-oriented design principles in developing a To-Do list application. Utilizing a classroom GitHub repository, I implemented features that enable users to add new tasks and filter existing ones based on search criteria. The project emphasizes encapsulation, modularity, and the use of classes to manage task data, reflecting a solid understanding of object-oriented programming concepts.

React Menu

Building upon prior in-class exercises, I developed a data-driven menu page using React. The core of this project is a React component designed to accept properties adhering to a specific data structure, enabling dynamic rendering of menu items. This structure allows for scalability and easy updates, demonstrating an understanding of component-based architecture and the importance of prop-driven data flows in React applications.

Expo To Do List

In this project, I developed a To-Do list application using Expo and React Native. The app offers a user-friendly interface for adding and completing tasks, leveraging React Native's FlatList component for efficient rendering of task lists. By integrating various packages and dependencies, the application ensures smooth performance and a native-like experience on mobile devices. This project highlights my ability to utilize React Native components and manage dependencies effectively in a mobile development environment.
